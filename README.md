2. Prompt Engineering Frameworks

I utilize a structured approach to prompting LLMs (Cursor/Claude/GPT-4) to ensure consistent outputs for business operations.

My Standard Prompt Structure:

Role: Define the persona (e.g., "Senior Python Architect" or "Legal Compliance Officer").

Context: Provide background data and constraints.

Task: Specific, actionable instruction.

Output Format: JSON, Markdown table, or bullet points.

Refinement: Iterative feedback loop based on edge cases.
